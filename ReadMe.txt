	CONSOLE KEEPER


Instructions de lancement
	Ouvrir la console dans le dossier du jeu puis executer dans la console:
	java -jar ConsoleKeeper.jar

Dépandances
	il est nécessaire d'avoir installé une version à jour du JDK (java developpement kit).

	le JDK est disponible ici:
	https://www.oracle.com/fr/java/technologies/downloads/
